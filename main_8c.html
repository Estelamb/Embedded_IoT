Detailed Description

Plant Monitoring System main module.

This module monitors plant conditions such as light, soil moisture, temperature/humidity, acceleration, color, and GPS location. It provides visual feedback through an RGB LED and allows the user to switch operating modes via a button.

Operating Modes:

  • TEST_MODE: RGB LED shows the dominant color detected.
  • NORMAL_MODE: Periodic measurements; RGB LED alerts if any sensor is out of range.
  • ADVANCED_MODE: Periodic measurements; RGB LED shows real color.

Button Behavior:

  • Toggles between TEST, NORMAL, and ADVANCED modes with each press.

enum system_mode_t

System operating modes.

These modes define how the system behaves:

  • TEST_MODE: Shows the dominant detected color through the RGB LED.
  • NORMAL_MODE: Periodically measures sensors and alerts if any reading is out of range.
  • ADVANCED_MODE: Emulates PWM for RGB LED control.

int main ( void  )

Main entry point for the brightness control system.

Initializes peripherals (RGB LED, ADC, user button), starts the brightness thread, and executes the LED update loop.

Button input is interrupt-driven; all press logic is handled by ISR and workqueue.

Returns
This function does not return under normal operation.
